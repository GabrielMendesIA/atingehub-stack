## Pré-requisito

Você precisa do **Claude Code** instalado.

Se não tem: https://claude.com/claude-code

---

## Instalação (2 comandos)

Dentro do Claude Code, rode:

```
/plugin marketplace add gabrielmendes/atingehub-stack
```

Depois:

```
/plugin install atingehub-stack@atingehub
```

Pronto. As skills aparecem automaticamente — basta pedir tarefas relacionadas (escrever copy, criar imagem, fazer pesquisa de cliente, etc) que o Claude vai usar a skill certa.

---

## Atualizar

Sempre que o Gabriel publicar versão nova:

```
/plugin marketplace update atingehub
```

---

## Como funciona

Esse repositório é **espelho periódico** do setup pessoal do Gabriel. Quando ele cria skill nova ou ajusta agente existente, ele roda um script de sincronização que atualiza esse repo (sem incluir nada pessoal — sem credenciais, sem memory, sem skills proprietárias).

Você sempre está alinhado com o estado mais recente da consultoria.
